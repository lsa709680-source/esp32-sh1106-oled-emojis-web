# ESP32 + SH1106 0.96" OLED（128×64）18种表情包 + 网页切换

这是一个轻量、可直接上手的 **ESP32 + SH1106 128×64 OLED** 项目：内置 **18 种表情**（表情包），并提供一个 **网页控制界面**，用手机/电脑浏览器即可一键切换表情。  
开发环境为 **Arduino IDE**，适合做桌面机器人、伴生宠、设备状态提示、互动小装置等。

---

## ✅ 功能特点
- ✅ 支持 SH1106 0.96 寸 OLED（128×64）
- ✅ 内置 18 种表情（可扩展）
- ✅ ESP32 内置网页控制：浏览器一键切换表情
- ✅ 代码结构清晰，适合新手二次开发
- ✅ 可用于机器人“眼睛/情绪/状态显示”

---

## 🧰 硬件清单
- ESP32（DevKit / WROOM 等，其他型号可自行适配）
- 1.3" OLED 128×64（驱动芯片：**SH1106**）
- I2C 连接线（SDA/SCL/VCC/GND）

> 注意：市面上也有 SSD1306 的 128×64 OLED，外观很像。  
> 本仓库面向 **SH1106**。如果你的屏幕是 SSD1306，通常只需要改一下驱动构造方式/初始化代码。

---

## 🔌 接线方式（I2C）
| OLED 引脚 | ESP32 |
|----------|------|
| VCC      | 3.3V |
| GND      | GND  |
| SDA      | GPIO21（默认） |
| SCL      | GPIO22（默认） |

> 如果你用的不是 21/22，请在代码里修改 `Wire.begin(SDA, SCL)`。

## 🚀 快速开始
### 1）安装依赖库
请在 Arduino IDE 的库管理器中安装（或按你实际使用填写）：
- `U8g2`（推荐）
- ESP32 自带：`WiFi` / `WebServer`（如果你的代码使用了网页控制）

> 你也可以把你实际用的库名写在这里（更准确）。

### 2）选择开发板并上传
- 开发板：选择你的 ESP32 对应型号（如 ESP32 Dev Module）
- 端口：选择正确 COM 口
- 上传代码到 ESP32

### 3）打开网页控制
- 如果代码开启了热点/AP：连接 ESP32 的 WiFi 热点  
- 浏览器打开项目提示的地址（示例：`192.168.4.1`）  
- 点击网页按钮，即可切换 OLED 表情

## 🛠️ 自定义与扩展
你可以很容易地：
- 替换/新增表情图案（位图数组）
- 增加网页按钮
- 调整显示方向/偏移（解决镜像或左右偏移）

常见问题排查：
- 屏幕不亮：检查 I2C 地址（常见 `0x3C`）、接线、电压
- 显示偏移：SH1106 常见需要设置 x-offset 或旋转参数
- 显示镜像：调整 `setFlipMode` / `setRotation`（取决于你使用的库）
